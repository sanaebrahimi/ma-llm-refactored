Usage
=====

Run from the ``refactored`` folder.

Dynamic + chain runs
--------------------

- QA: ``python multi-agent_coordinator.py``
- MATH/GSM8K: ``python multi_agent_coordinator_MATH.py``
- MMLU: ``python multi_agent_coordinator_MMLU.py``
- HumanEval: ``python multi_agent_coordinator_HumanEval.py``

Post-hoc judging
----------------

- ``python judge.py``
